# diy-pizza
A simple web app for customizing your pizza! 

## To get started

### 1. Install requirements

#### NodeJS 8.x.x

Windows installer: https://nodejs.org/en/download/

Install instructions for Linux: https://nodejs.org/en/download/package-manager/

The project was tested using NodeJS 8.16.12. Feel free to use other versions at your own risk.

### 2. Clone this repository
Run the below command to the directory where the repository will be located.
```
git clone git@github.com:kylakhan/diy-pizza.git
```

### 3. Install dependencies

```
cd <repository directory>
npm install
cd server
npm install
cd ../client
npm install
```
### 4. Start the server and client

The below command will start both the client and the server concurrently.
The client and server are hardcoded to listen to ports 3000 and 3001, respectively.

```
cd <repository directory>
npm start
```

### 5. Open a web browser
The web app should now be accessible at http://localhost:3000


Note: All icon links for the pizza toppings are from https://icons8.com.
